# itmo-advanced-ML

Курс посвящен углубленному изучению методов машинного обучения и нейронных сетей.  
В рамках курса реализованы модели глубокого обучения, проведены эксперименты с архитектурами, функциями активации, оптимизаторами и стратегиями обучения.

## Лабораторная работа 1 — Нелинейная регрессия

Сгенерированы синтетические нелинейные функции и обучена нейросеть для их аппроксимации.  
Проведено сравнение влияния:
- размера батча  
- batch normalization  
- dropout  

Результаты визуализированы, выполнен анализ качества моделей.

[Аппроксимация нелинейных функций](./advanced-ml/lab1/advanced_ml_lab1.ipynb)

## Лабораторная работа 2 — Классификация изображений

Решена задача классификации цветов автомобилей на датасете DVM.  
Реализованы и обучены модели:
- ResNet / DenseNet / MobileNet (с нуля)  
- предобученные модели (fine-tuning)

Проведена оценка качества по метрике **F1-macro** и сравнение моделей.

[Классификация изображений автомобилей](./advanced-ml/lab2/advanced_ml_lab2.ipynb)

## Задание 1 — MNIST и функции активации

Реализована нейронная сеть (MLP / CNN) для классификации изображений MNIST.  
Проведено сравнение функций активации:
- ReLU  
- GELU  
- Swish  
- Softplus  

Построены графики зависимости точности от эпох.

[Сравнение функций активации](./advanced-ml/task1/advanced_ml_task1.ipynb)

- SGD / Adam / RMSProp  

А также стратегии изменения learning rate:

Проведен анализ сходимости и визуализация изменения learning rate.
