# Лабораторная 2 — Классификация изображений

В рамках работы решена задача классификации цветов автомобилей на датасете DVM.

## Что сделано
- реализованы и обучены модели компьютерного зрения;
- проведено сравнение архитектур ResNet / DenseNet / MobileNet;
- выполнен fine-tuning предобученных моделей;
- оценено качество по метрике F1-macro.
