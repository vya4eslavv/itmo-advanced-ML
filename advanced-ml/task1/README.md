# Задание 1 — MNIST и функции активации

В рамках задания реализована нейронная сеть для классификации изображений рукописных цифр MNIST.

## Что сделано
- обучена MLP или CNN-модель на датасете MNIST;
- реализовано сравнение функций активации ReLU, GELU, Swish и Softplus;
- построены графики зависимости точности от эпох;
- выполнен анализ влияния функций активации на обучение
