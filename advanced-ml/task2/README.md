# Задание 2 — Оптимизаторы и learning rate

В рамках задания исследованы методы оптимизации и стратегии изменения learning rate.

## Что сделано
- реализованы и протестированы оптимизаторы SGD, Adam и RMSProp;
- исследованы scheduler'ы StepLR и ExponentialLR;
- построены графики изменения learning rate;
- проанализирована сходимость моделей при разных настройках обучения.
